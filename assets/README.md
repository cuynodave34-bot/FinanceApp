# App Assets

Place final image assets in this folder before building for production.

## Required assets

| File | Size | Purpose |
|------|------|---------|
| `icon.png` | 1024 × 1024 | App store and home-screen icon |
| `splash.png` | 1242 × 2436 | Launch screen background |
| `adaptive-icon.png` | 1024 × 1024 | Android adaptive icon foreground |
| `favicon.png` | 48 × 48 | Web favicon |

## Asset guidelines

- Use the app canvas color `#f4efe3` as the dominant background so the launch screen feels seamless.
- Keep the icon centered with safe margins; both iOS and Android will mask it into rounded shapes.
- Export splash as a solid background with a small centered logo or wordmark.
